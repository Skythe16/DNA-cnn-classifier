# DNA Sequence Classification using CNN

## Overview
This project uses a convolutional neural network to classify DNA sequences.

## Tools Used
- Python
- TensorFlow / Keras
- Jupyter Notebook

## What I Did
- Preprocessed DNA sequence data using one-hot encoding
- Built and trained a CNN model for binary classification
- Tuned hyperparameters to improve performance
- Reached about 84% test accuracy

## Files
- `dna_cnn_classifier.ipynb` - main notebook containing preprocessing, model building, training, and evaluation
- `dm3.kc167.tads.train.csv` - training dataset
- `dm3.kc167.tads.test.csv` - test dataset
